## ✨ Destaques do Projeto

- **Dashboard Financeiro Completo**: Visão centralizada de Saldo, Receitas, Despesas, Metas e Investimentos.
- **PDV (Ponto de Venda)**: Interface ágil para vendas, com atalhos de teclado, descontos e suspensão de vendas.
- **Módulo Fiscal (NFC-e)**: Emissão de Nota Fiscal de Consumidor Eletrônica com geração de XML e DANFE.
- **Controle de Estoque**: Gestão de produtos, fornecedores e movimentações de entrada/saída.
- **Investimentos Inteligentes**: 
    - Acompanhamento de ações e FIIs com integração ao **Yahoo Finance**.
    - Atualização em tempo real de Lucro/Prejuízo e variação percentual.
- **Relatórios Avançados**: 
    - Gráficos interativos de evolução mensal, distribuição de despesas e performance de vendas.
    - Exportação profissional em **PDF** e **CSV**.
- **Portabilidade Total**: Versão compilada em `.exe` único que roda direto do pendrive ou pasta, sem instalar nada.

## ⚡ Automação (Scripts Facilitadores)

Para facilitar o uso, incluímos scripts automáticos (`.bat`) na raiz do projeto. Basta clicar duas vezes:

| Arquivo | Função |
|---------|--------|
| `install.bat` | **Instalação Completa**: Configura Python, cria ambiente virtual e instala dependências. |
| `run.bat` | **Rodar Projeto**: Inicia o servidor local e abre o navegador. |
| `build.bat` | **Criar Executável**: Gera o arquivo `.exe` (incluindo `staticfiles` e configurações atualizadas) na pasta `dist`. |
| `populate_data.bat` | **Dados de Exemplo**: Cria usuário teste e transações fictícias. |
| `backup.bat` | **Backup**: Cria um arquivo `.zip` com todo o código do projeto. |

## 🔧 Instalação e Desenvolvimento (Código Fonte)

Se você é desenvolvedor e quer alterar o código, siga estes passos:

### Pré-requisitos
- Python 3.10 ou superior
- Git

### Passo a Passo

```bash
# 1. Clone o repositório
git clone https://github.com/seu-usuario/finance-project.git
cd finance-project

# 2. Crie e ative o ambiente virtual
python -m venv venv
# Windows:
venv\Scripts\activate
# Linux/Mac:
source venv/bin/activate

# 3. Instale as dependências
pip install -r requirements.txt

# 4. Configure o Banco de Dados
python manage.py migrate

# 5. Execute o Servidor
python manage.py runserver
```

**🌐 Acesse:** `http://localhost:8000`
