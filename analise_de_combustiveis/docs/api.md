# API

## Endpoints

- `GET /health`
- `GET /api/v1/fuels`
- `GET /api/v1/overview?fuel=gasolina&state=SP`
- `GET /api/v1/history?fuel=etanol&state=GO&city=Goiania`
- `GET /api/v1/forecast?fuel=gasolina&state=SP&horizon=12`
- `GET /api/v1/compare?fuel=gasolina&compare_with=etanol&state=SP`
- `GET /api/v1/map?fuel=diesel`
- `GET /api/v1/insights?fuel=gasolina&state=SP`

## Resposta base

As respostas seguem JSON com estrutura enxuta, pronta para consumo do frontend.
